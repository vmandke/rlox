# Lox Demo Programs

> **Note:** All programs in this directory were generated with the assistance of
> [Claude Code](https://claude.ai/code).

Run any program with:
```
cargo run -- progs/<name>.lox
```

All visual programs use **fixed-point integer arithmetic** (scale = 1000) and
**double-bracket modulo** `n - ((n/d)*d)` to work around integer division precedence.
